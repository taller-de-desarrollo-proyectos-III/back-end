Up 
get /volunteers
(volunteersGet)
by passing in the appropriate options, you can get all the existing volunteers

Query parameters

commissionUuids (required)
Query Parameter — used for filters. Default is ALL or can be the commission uuid you are looking for
roleUuids (required)
Query Parameter — used for filters. Default is ALL or can be the role uuid you are looking for
stateUuids (required)
Query Parameter — used for filters. Default is ALL or can be the state uuid you are looking for

Responses

200

search results matching criteria

400

bad input parameter
